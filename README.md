# Poinpy-like:
Poinpy is a mobile game published by Netflix. I do not have Netflix or access to a Netflix account required to play the game, so I figured I'd make my own Poinpy-clone, tweaked to more closely match my own preferences.

# Core Systems:
Much like Poinpy, the game involves flicking a ball up a vertical shaft, richoceting off walls to gain greater height.

# Design & Programming Goals:
I want to emphasize a snowballing effect for the player, and how that contrast against the rubber banding tension of the chaser enemy following after you.

# Technologies & Architecture:
- Unity (C#): Core engine used for 3D rendering, input, and system management;

# Current Status & Future Plans:
This game is still very early days as university priorities overtook those of the project.
I do plan on finishing it one day, implementing more of Poinpy's design and playing around more to experiement with my own additions.
